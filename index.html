Sublime Text 2 tipJS snippets

Code snippets for developing with tipJS.com – Open Source JavaScript MVC Framework. Sublime Text 2 에서 tipJS 를 쉽게 사용하기 위한 스니펫 모음입니다. 현재는 config, define, model, controller, view 를 지원합니다.

Installation

현재 wbond's Package Control 에서 설치 할 수 있습니다.

Without Package Control

Mac OS

cd "~/Library/Application Support/Sublime Text 2/Packages"

Windows 7

cd "C:\Users\YOUR_USERNAME\AppData\Roaming\Sublime Text 2\Packages"

Inside Packages 

git clone https://github.com/rkjun/sublime-tipjs-snippets.git "tipJS"
